### settingup
```
git clone https://github.com/mcmc101001/codesprint_psa
python -m pip install --upgrade pip
pip3 install django
```
### git commands
```
git pull
git add .
git commit -m "message"
git push
```

### running server
```
python manage.py runserver
```

### modifying models
```
python manage.py makemigrations
python manage.py migrate
```
